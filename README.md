### Prerequis
- [Node.js](https://nodejs.org/en/) > v16  
- [MongoDB Community Server](https://www.mongodb.com/try/download/community)

### Installation
- cloner / forker le repo  
- npm install  
- npm start  

Le server se lance sur le port 5000  
La base de données est visible dans mongoDB
